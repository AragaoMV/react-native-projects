
Esse App é uma aplicação feita em React Native que simula um e-commerce. A ideia é usar o Context API, do react, para ver a vantagem de se ter estados globais na aplicação. Também é usado o AsyncStorage e uma Web API para ter a persistência dos dados.

Esse projeto é utilizado na formação base de React Native da plataforma da Alura.

Tecnologias

Esse projeto foi desenvolvido com a utilização das seguintes tecnologias:
- React Native
- Expo
